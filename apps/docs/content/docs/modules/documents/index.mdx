## Métodos disponibles

| Método                           | Método HTTP + Endpoint                      | Descripción                                                          |
| -------------------------------- | ------------------------------------------- | -------------------------------------------------------------------- |
| `downloadXml(trackId, options?)` | `GET /v2/documents/{track_id}/download-xml` | Descargar el XML de un documento a partir de su identificador global |

Usa este módulo cuando ya tienes el identificador global del documento y no necesitas navegar por un módulo específico como `bills`, `creditNotes` o `supportDocuments`.

`trackId` puede corresponder a distintos identificadores definidos por la DIAN, por ejemplo:

- `CUFE` para facturas electrónicas
- `CUDE` para notas crédito o débito
- `CUDS` para documentos soporte
- `CUNE` para nómina electrónica
- Otros identificadores equivalentes definidos por la DIAN
