---
title: Suscripciones
description: Obtener todas las suscripciones activas de la cuenta.
---

## Firma del método

```ts
list(): Promise<ApiResponse<Subscription[]>>
```

## Endpoint Factus

`GET /v1/subscriptions`

Fuente oficial: [Suscripciones](https://developers.factus.com.co/v1/suscripciones/suscripciones)

## Parámetros

Este método no recibe parámetros.

## Ejemplo

```ts
const response = await factus.subscription.list();
const subscriptions = response.data;

for (const subscription of subscriptions) {
  console.log(subscription.name);
  console.log(subscription.documents_quota);
  console.log(subscription.documents_consumed);
  console.log(subscription.documents_available);
}
```
