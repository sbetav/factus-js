## Firma


## Endpoint Factus

`GET /v1/subscriptions/current`

Fuente oficial: [Suscripción actual](https://developers.factus.com.co/suscripciones/suscripcion-actual.html)

## Ejemplo

```ts
const response = await factus.subscription.getCurrent();

console.log(response.data.total_documents);
console.log(response.data.documents_used);
console.log(response.data.documents_remaining);
```
