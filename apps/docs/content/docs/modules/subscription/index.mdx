Accede al modulo a traves de `factus.subscription`.

<Callout type="info">
  Esta sección sigue los endpoints oficiales de Factus:
  [Suscripciones](https://developers.factus.com.co/suscripciones/suscripciones.html)
  y [Suscripción actual
  (deprecated)](https://developers.factus.com.co/suscripciones/suscripcion-actual.html).
</Callout>

## Métodos disponibles

| Método         | Verbo + Ruta                    | Descripción                                |
| -------------- | ------------------------------- | ------------------------------------------ |
| `list()`       | `GET /v1/subscriptions`         | Listar suscripciones activas (recomendado) |
| `getCurrent()` | `GET /v1/subscriptions/current` | Resumen legado de la suscripción actual    |
