---
title: Descargar XML
description: Descargar XML de una nota de ajuste en base64 por numero.
---

## Firma del método

```ts
downloadXml(number: string): Promise<ApiResponse<DownloadAdjustmentNoteXmlResponse>>
```

## Endpoint Factus

`GET /v1/adjustment-notes/download-xml/{number}`

Fuente oficial: [Descargar XML](https://developers.factus.com.co/notas-ajuste-documentos-soporte/descargar-xml/index.html)

## Parámetros

<TypeTable
  type={{
    number: {
      type: "string",
      required: true,
      description:
        "Número del documento de la nota de ajuste cuyo XML se descargará.",
    },
  }}
/>
## Ejemplo

```ts
const response = await factus.adjustmentNotes.downloadXml("NDA35");
const xml = Buffer.from(response.data.xml_base_64_encoded, "base64").toString(
  "utf-8",
);
```
