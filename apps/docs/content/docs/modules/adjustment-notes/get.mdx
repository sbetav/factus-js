---
title: Ver nota de ajuste
description: Obtener detalle de una nota de ajuste por numero.
---

## Firma del método

```ts
get(
  number: string,
  options?: RequestOptions,
): Promise<ViewAdjustmentNoteResponse>
```

## Endpoint Factus

`GET /v2/adjustment-notes/{number}`

Fuente oficial: [Ver nota de ajuste](https://developers.factus.com.co/notas-ajuste-documentos-soporte/ver)

## Parámetros

<TypeTable
  type={{
    number: {
      type: "string",
      required: true,
      description: "Número del documento de la nota de ajuste.",
    },
  }}
/>
## Ejemplo

```ts
const response = await factus.adjustmentNotes.get("NDA35");
console.log(response.data);
```
