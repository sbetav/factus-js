---
title: Descargar PDF
description: Descargar PDF de una nota de ajuste en base64 por numero.
---

## Firma del método

```ts
downloadPdf(number: string): Promise<ApiResponse<DownloadAdjustmentNotePdfResponse>>
```

## Endpoint Factus

`GET /v1/adjustment-notes/download-pdf/{number}`

Fuente oficial: [Descargar PDF](https://developers.factus.com.co/notas-ajuste-documentos-soporte/descargar-nota-ajuste-soporte-pdf/index.html)

## Parámetros

<TypeTable
  type={{
    number: {
      type: "string",
      required: true,
      description:
        "Número del documento de la nota de ajuste cuyo PDF se descargará.",
    },
  }}
/>
## Ejemplo

```ts
const response = await factus.adjustmentNotes.downloadPdf("NDA35");
const pdfBuffer = Buffer.from(response.data.pdf_base_64_encoded, "base64");
```
