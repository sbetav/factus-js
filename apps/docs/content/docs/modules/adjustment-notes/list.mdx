## Firma del método

```ts
list(
  params?: ListParams<AdjustmentNoteFilters>,
  options?: RequestOptions,
): Promise<GetAdjustmentNotesResponse>
```

## Endpoint Factus

`GET /v2/adjustment-notes`

Fuente oficial: [Listar notas de ajuste](https://developers.factus.com.co/notas-ajuste-documentos-soporte/ver-y-filtrar)

## Ejemplo

```ts
const response = await factus.adjustmentNotes.list({
  filter: { status: "1" },
  page: 1,
  per_page: 10,
});

const adjustmentNotes = response.data.data;
const pagination = response.data.pagination;
```
