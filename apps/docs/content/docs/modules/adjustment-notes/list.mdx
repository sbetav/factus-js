## Firma del método

```ts
list(filters?: AdjustmentNoteFilters & { page?: number; per_page?: number }): Promise<ApiResponse<PaginatedData<AdjustmentNote>>>
```

## Endpoint Factus

`GET /v1/adjustment-notes`

Fuente oficial: [Filtrar notas de ajuste](https://developers.factus.com.co/notas-ajuste-documentos-soporte/ver-y-filtrar/index.html)

## Ejemplo

```ts
const response = await factus.adjustmentNotes.list({
  "filter[status]": "1",
  page: 1,
  per_page: 10,
});

const adjustmentNotes = response.data.data;
const pagination = response.data.pagination;
```
