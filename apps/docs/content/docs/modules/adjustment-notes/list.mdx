## Firma del método

```ts
list(params?: ListParams<AdjustmentNoteFilters>): Promise<ApiResponse<PaginatedData<AdjustmentNote>>>
```

## Endpoint Factus

## Ejemplo

```ts
const response = await factus.adjustmentNotes.list({
  filter: { status: "1" },
  page: 1,
  per_page: 10,
});

const adjustmentNotes = response.data.data;
const pagination = response.data.pagination;
```
