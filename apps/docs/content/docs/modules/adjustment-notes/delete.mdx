import { Callout } from "fumadocs-ui/components/callout";

## Firma del método

```ts
delete(referenceCode: string): Promise<DeleteAdjustmentNoteResponse>
```

## Endpoint Factus

`DELETE /v1/adjustment-notes/reference/{reference_code}`

Fuente oficial: [Eliminar nota de ajuste](https://developers.factus.com.co/notas-ajuste-documentos-soporte/eliminar/index.html)

<Callout type="warn">
  Solo se pueden eliminar notas de ajuste que no hayan sido validadas por la
  DIAN.
</Callout>

## Parámetros

<TypeTable
  type={{
    "referenceCode": {
      type: "string",
      required: true,
      description: <>{"Código de referencia ("}<code>{"reference_code"}</code>{") de la nota de ajuste no validada a eliminar."}</>,
    },
  }}
/>
## Ejemplo

```ts
await factus.adjustmentNotes.delete("REF007");
```

  Es probable que este método no funcione en entorno sandbox.
