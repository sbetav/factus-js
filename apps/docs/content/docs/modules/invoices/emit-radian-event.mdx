import { Callout } from "fumadocs-ui/components/callout";

## Firma del método

```ts
emitRadianEvent(
  number: string,
  eventType: EventCode,
  input: RadianEventUpdateInput,
): Promise<RadianEventUpdateResponse>
```

## Endpoint Factus

`POST /v1/bills/radian/events/update/{number}/{event_type}`

Fuente oficial: [Aceptación tacita](https://developers.factus.com.co/facturas/aceptacion-tacita/index.html)

## Tipos de evento (`EventCode`)

- `RECEIPT_ACKNOWLEDGEMENT` (`"030"`)
- `INVOICE_CLAIM` (`"031"`)
- `GOODS_OR_SERVICE_RECEIPT` (`"032"`)
- `EXPRESS_ACCEPTANCE` (`"033"`)
- `TACIT_ACCEPTANCE` (`"034"`)

## Ejemplo

```ts
import { EventCode, IdentityDocumentTypeId } from "factus-js";

await factus.invoices.emitRadianEvent(
  "SETP990000001",
  EventCode.TACIT_ACCEPTANCE,
  {
    identification_document_code: IdentityDocumentTypeId.CITIZENSHIP_ID,
    identification: "12345678",
    first_name: "Pepito",
    last_name: "Perez",
    job_title: "Compras",
    organization_department: "Administracion",
  },
);
```

<Callout type="info">
  Este método cubre el flujo documentado por Factus para emisión de eventos
  RADIAN (incluyendo aceptación tacita, según el `eventType`).
</Callout>
