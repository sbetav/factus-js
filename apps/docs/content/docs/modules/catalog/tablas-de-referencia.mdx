Esta página consolida las tablas oficiales de Factus y su equivalente en
constantes de `factus-js`.

Fuente oficial:
[Tablas de referencia](https://developers.factus.com.co/tablas-de-referencia/tablas/index.html)

## Uso

Si instalaste `factus-js`, las constantes ya están disponibles.

```ts
import { PaymentFormCode, EventCode } from "factus-js";
```

<Callout type="info">
  Estas tablas no se consultan por HTTP: se exponen como constantes tipadas.
  Para datos dinámicos (países, municipios, tributos, adquirientes), usa los
  demás métodos de `factus.catalog`.
</Callout>

## Constantes disponibles

- `InvoiceDocumentType`
- `OperationTypeCode`
- `CreditNoteOperationTypeCode`
- `CreditNoteCorrectionCode`
- `ProductStandardId`
- `ClaimConceptCode`
- `EventCode`
- `IdentityDocumentTypeId`
- `SupportDocumentIdentityTypeId`
- `CustomerTributeId`
- `OrganizationTypeId`
- `PaymentMethodCode`
- `PaymentFormCode`
- `NumberingRangeDocumentTypeCode`
- `AdjustmentNoteReasonCode`
- `ChargeDiscountCode`
- `FiscalResponsibilityCode`

## Ejemplo

```ts
import {
  EventCode,
  IdentityDocumentTypeId,
  NumberingRangeDocumentTypeCode,
  PaymentMethodCode,
} from "factus-js";

const eventType = EventCode.EXPRESS_ACCEPTANCE;
const idType = IdentityDocumentTypeId.NIT;
const documentType = NumberingRangeDocumentTypeCode.SALES_INVOICE;
const paymentMethod = PaymentMethodCode.TRANSFER;
```
