---
title: Información adquirientes
description: Consultar nombre y correo del adquiriente por tipo y numero de documento.
---

## Firma

```ts
getAcquirer(filters: AcquirerFilters): Promise<ApiResponse<Acquirer>>
```

## Endpoint Factus

`GET /v1/dian/acquirer`

Fuente oficial: [Obtener datos de adquiriente](https://developers.factus.com.co/informacion-adquirientes/obtener-datos-adquiriente/index.html)

## Parámetros

- `identification_document_id`
- `identification_number`

## Ejemplo

```ts
import { IdentityDocumentTypeId } from "factus-js";

const response = await factus.catalog.getAcquirer({
  identification_document_id: IdentityDocumentTypeId.NIT,
  identification_number: "3199995",
});

console.log(response.data.name);
console.log(response.data.email);
```
