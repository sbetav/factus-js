Accede al modulo a traves de `factus.catalog`.

<Callout type="info">
  Esta sección consolida endpoints de consulta maestra y referencias usadas por
  todos los documentos electrónicos.
</Callout>

## Métodos disponibles

| Método                           | Método HTTP + Endpoint      | Descripción                                                         |
| -------------------------------- | --------------------------- | ------------------------------------------------------------------- |
| `getAcquirer(filters)`           | `GET /v1/dian/acquirer`     | Obtener nombre/email del adquiriente por tipo y número de documento |
| `listTributes(filters?)`         | `GET /v1/tributes/products` | Listar tributos de productos                                        |
| `listMeasurementUnits(filters?)` | `GET /v1/measurement-units` | Listar unidades de medida                                           |
| `listCountries(params?)`         | `GET /v1/countries`         | Listar países                                                       |
| `listMunicipalities(params?)`    | `GET /v1/municipalities`    | Listar municipios                                                   |

## Tablas de referencia

Las tablas de referencia (tipos de documento, eventos, formas de pago, etc.) se
exponen como constantes en `factus-js` y no requieren
llamadas HTTP.
