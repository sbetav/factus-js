## Firma del método

```ts
listCountries(filters?: CountryFilters & { page?: number; per_page?: number }): Promise<ApiResponse<Country[]>>
```

## Endpoint Factus

`GET /v1/countries`

Fuente oficial: [Obtener países](https://developers.factus.com.co/paises/obtener-paises/index.html)
