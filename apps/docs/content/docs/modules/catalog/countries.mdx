---
title: Países
description: Listar países disponibles para datos de terceros y empresa.
---

## Firma del método

```ts
listCountries(params?: ListParams<CountryFilters>): Promise<ApiResponse<Country[]>>
```

## Endpoint Factus

`GET /v1/countries`

Fuente oficial: [Obtener países](https://developers.factus.com.co/v1/paises/obtener-paises/index.html)

## Ejemplo

```ts
const response = await factus.catalog.listCountries({
  filter: {
    name: "Colombia",
  },
});

console.log(response.data[0]);
```
