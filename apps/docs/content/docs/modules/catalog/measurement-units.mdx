## Firma del método

```ts
listMeasurementUnits(filters?: MeasurementUnitFilters): Promise<ApiResponse<MeasurementUnit[]>>
```

## Endpoint Factus

Fuente oficial: [Obtener unidades](https://developers.factus.com.co/v1/unidades-de-medida/obtener-unidades/index.html)
