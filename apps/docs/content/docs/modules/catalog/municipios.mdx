---
title: Municipios
description: Listar municipios disponibles para direcciones y ubicaciones.
---

## Firma del método

```ts
listMunicipalities(filters?: MunicipalityFilters & { page?: number; per_page?: number }): Promise<ApiResponse<Municipality[]>>
```

## Endpoint Factus

`GET /v1/municipalities`

Fuente oficial: [Obtener municipios](https://developers.factus.com.co/municipios/obtener-municipios/index.html)

## Filtros

- `name`

## Ejemplo

```ts
const response = await factus.catalog.listMunicipalities({
  name: "San Gil",
});

console.log(response.data[0]);
```
