## Firma del método

```ts
listTributes(filters?: TributeFilters): Promise<ApiResponse<Tribute[]>>
```

## Endpoint Factus

`GET /v1/tributes/products`

Fuente oficial: [Tributos de productos](https://developers.factus.com.co/tributos/tributos-de-productos/index.html)

## Ejemplo

```ts
const response = await factus.catalog.listTributes({
  name: "IVA",
});

for (const tributo of response.data) {
  console.log(tributo.id, tributo.code, tributo.name);
}
```
