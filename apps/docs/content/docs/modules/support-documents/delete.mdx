title: Eliminar documento soporte
description: Eliminar un documento soporte no validado por reference_code.
---

import { Callout } from "fumadocs-ui/components/callout";

## Firma del método

```ts
delete(referenceCode: string): Promise<DeleteSupportDocumentResponse>
```

## Endpoint Factus

`DELETE /v1/support-documents/reference/{reference_code}`

Fuente oficial: [Eliminar documento soporte](https://developers.factus.com.co/documentos-soporte/eliminar/index.html)

<Callout type="warn">
  Solo se pueden eliminar documentos soporte que no hayan sido validados por la
  DIAN.
</Callout>

## Ejemplo

```ts
await factus.supportDocuments.delete("REF0017");
```
