---
title: Descargar PDF
description: Descargar PDF de un documento soporte en base64 por numero.
---

## Firma del método

```ts
downloadPdf(
  number: string,
  options?: RequestOptions,
): Promise<DownloadSupportDocumentPdfResponse>
```

## Endpoint Factus

`GET /v2/support-documents/{number}/download-pdf`

Fuente oficial: [Descargar PDF](https://developers.factus.com.co/documentos-soporte/descargar-pdf)

## Parámetros

<TypeTable
  type={{
    number: {
      type: "string",
      required: true,
      description: "Número del documento soporte cuyo PDF se descargará.",
    },
  }}
/>
## Ejemplo

```ts
const response = await factus.supportDocuments.downloadPdf("SEDS984000085");
const pdfBuffer = Buffer.from(response.data.pdf_base_64_encoded, "base64");
```
