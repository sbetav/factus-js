## Métodos disponibles

| Método                            | Método HTTP + Endpoint                                    | Descripción                                |
| --------------------------------- | --------------------------------------------------------- | ------------------------------------------ |
| `create(input, options?)`         | `POST /v2/support-documents/validate`                     | Crear y validar un documento soporte       |
| `list(params?, options?)`         | `GET /v2/support-documents`                               | Listar documentos soporte                  |
| `listAll(filter?, options?)`      | `GET /v2/support-documents`                               | Recorrer todas las páginas automáticamente |
| `get(number, options?)`           | `GET /v2/support-documents/{number}`                      | Obtener el detalle del documento           |
| `downloadXml(number, options?)`   | `GET /v2/support-documents/{number}/download-xml`         | Descargar el XML firmado                   |
| `downloadPdf(number, options?)`   | `GET /v2/support-documents/{number}/download-pdf`         | Descargar el PDF                           |
| `delete(referenceCode, options?)` | `DELETE /v2/support-documents/reference/{reference_code}` | Eliminar un documento no validado          |

Consulta [Paginación automática](/docs/pagination) para el uso de `listAll()`.
