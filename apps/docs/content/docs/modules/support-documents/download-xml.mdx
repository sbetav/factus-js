---
title: Descargar XML
description: Descargar XML de un documento soporte en base64 por numero.
---

## Firma del método

```ts
downloadXml(
  number: string,
  options?: RequestOptions,
): Promise<DownloadSupportDocumentXmlResponse>
```

## Endpoint Factus

`GET /v2/support-documents/{number}/download-xml`

Fuente oficial: [Descargar XML](https://developers.factus.com.co/documentos-soporte/descargar-xml)

## Parámetros

<TypeTable
  type={{
    number: {
      type: "string",
      required: true,
      description: "Número del documento soporte cuyo XML se descargará.",
    },
  }}
/>
## Ejemplo

```ts
const response = await factus.supportDocuments.downloadXml("SEDS984000085");
const xml = Buffer.from(response.data.xml_base_64_encoded, "base64").toString(
  "utf-8",
);
```
