---
title: Crear documento soporte
description: Crear y validar un documento soporte electrónico con Factus API v2.
---

## Firma del método

```ts
create(
  input: CreateSupportDocumentInput,
  options?: RequestOptions,
): Promise<CreateSupportDocumentResponse>
```

## Endpoint Factus

`POST /v2/support-documents/validate`

Fuente oficial: [Crear documento soporte](https://developers.factus.com.co/documentos-soporte/crear-validar)

## Ejemplo

```ts
import {
  PaymentFormCode,
  PaymentMethodCode,
  ProductStandardCode,
  SupportDocumentIdentityDocumentCode,
} from "factus-js";

const response = await factus.supportDocuments.create({
  numbering_range_id: 148,
  reference_code: "REF0017",
  observation: "Compra a proveedor no obligado a facturar",
  payment_details: [
    {
      payment_form: PaymentFormCode.CashPayment,
      payment_method_code: PaymentMethodCode.Cash,
      amount: "200000.00",
    },
  ],
  provider: {
    identification: "123456789",
    names: "Alan Turing",
    address: "calle 1 # 2-68",
    email: "alanturing@enigmasas.com",
    phone: "1234567890",
    country_code: "CO",
    identification_document_code: SupportDocumentIdentityDocumentCode.NIT,
    municipality_code: "11001",
  },
  items: [
    {
      code_reference: "SRV-001",
      name: "Servicio prestado",
      quantity: "1.00",
      discount_rate: "0.00",

});
```
