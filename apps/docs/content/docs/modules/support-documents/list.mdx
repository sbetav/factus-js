## Firma del método

```ts
list(params?: ListParams<SupportDocumentFilters>): Promise<ApiResponse<PaginatedData<SupportDocument>>>
```

## Endpoint Factus

`GET /v1/support-documents`

Fuente oficial: [Filtrar documentos soporte](https://developers.factus.com.co/v1/documentos-soporte/ver-y-filtrar/index.html)

## Ejemplo

```ts
const response = await factus.supportDocuments.list({
  filter: { status: "1" },
  page: 1,
  per_page: 10,
});

const supportDocuments = response.data.data;
const pagination = response.data.pagination;
```
