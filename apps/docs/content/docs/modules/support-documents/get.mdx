---
title: Ver documento soporte
description: Obtener detalle de un documento soporte por numero.
---

## Firma del método

```ts
get(number: string): Promise<ApiResponse<ViewSupportDocumentData>>
```

## Endpoint Factus

`GET /v1/support-documents/show/{number}`

Fuente oficial: [Ver documento soporte](https://developers.factus.com.co/v1/documentos-soporte/ver/index.html)

## Parámetros

<TypeTable
  type={{
    number: {
      type: "string",
      required: true,
      description: "Número del documento soporte a consultar.",
    },
  }}
/>
## Ejemplo

```ts
const response = await factus.supportDocuments.get("SEDS984000085");
console.log(response.data.support_document.total);
```
