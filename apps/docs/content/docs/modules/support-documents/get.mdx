---
title: Ver documento soporte
description: Obtener detalle de un documento soporte por numero.
---

## Firma del método

```ts
get(
  number: string,
  options?: RequestOptions,
): Promise<ViewSupportDocumentResponse>
```

## Endpoint Factus

`GET /v2/support-documents/{number}`

Fuente oficial: [Ver documento soporte](https://developers.factus.com.co/documentos-soporte/ver)

## Parámetros

<TypeTable
  type={{
    number: {
      type: "string",
      required: true,
      description: "Número del documento soporte a consultar.",
    },
  }}
/>
## Ejemplo

```ts
const response = await factus.supportDocuments.get("SEDS984000085");
console.log(response.data.number);
console.log(response.data.totals?.total);
```
