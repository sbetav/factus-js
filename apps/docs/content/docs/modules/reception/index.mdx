## Métodos disponibles

| Método                               | Método HTTP + Endpoint                                            | Descripción                                   |
| ------------------------------------ | ----------------------------------------------------------------- | --------------------------------------------- |
| `list(params?, options?)`            | `GET /v2/receptions/bills`                                        | Listar facturas recibidas                     |
| `listAll(filter?, options?)`         | `GET /v2/receptions/bills`                                        | Recorrer todas las páginas automáticamente    |
| `upload(input, options?)`            | `POST /v2/receptions/upload`                                      | Registrar una factura recibida por `track_id` |
| `emitEvent(params, input, options?)` | `PATCH /v2/receptions/bills/{bill_id}/radian/events/{event_type}` | Emitir un evento RADIAN manual (`030`-`033`)  |

Consulta [Paginación automática](/docs/pagination) para el uso de `listAll()`.
