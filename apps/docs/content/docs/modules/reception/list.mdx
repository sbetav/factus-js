---
title: Ver facturas
description: Listar y filtrar facturas electrónicas recibidas.
---

## Firma del método

```ts
list(filters?: ReceptionBillFilters & { page?: number; per_page?: number }): Promise<ApiResponse<PaginatedData<ReceptionBill>>>
```

## Endpoint Factus

`GET /v1/receptions/bills`

Fuente oficial: [Ver facturas](https://developers.factus.com.co/recepcion-de-documentos/ver-facturas/index.html)

## Filtros

- `filter[id]`
- `filter[number]`
- `filter[issue_date]`
- `filter[cufe]`
- `filter[company_nit]`
- `filter[company_name]`
- `filter[completed_events]`

## Ejemplo

```ts
const response = await factus.reception.list({
  "filter[completed_events]": "0",
  page: 1,
  per_page: 10,
});

const receptionBills = response.data.data;
const pagination = response.data.pagination;
```
