## Firma del método

```ts
list(params?: ListParams<ReceptionBillFilters>): Promise<ApiResponse<PaginatedData<ReceptionBill>>>
```

## Endpoint Factus

`GET /v1/receptions/bills`

Fuente oficial: [Ver facturas](https://developers.factus.com.co/v1/recepcion-de-documentos/ver-facturas/index.html)

## Ejemplo

```ts
const response = await factus.reception.list({
  filter: { completed_events: "0" },
  page: 1,
  per_page: 10,
});

const receptionBills = response.data.data;
const pagination = response.data.pagination;
```
