---
title: Ver empresa
description: Obtener los datos del perfil de la empresa autenticada.
---

## Firma

```ts
get(): Promise<ApiResponse<Company>>
```

## Endpoint Factus

`GET /v1/company`

Fuente oficial: [Ver empresa](https://developers.factus.com.co/empresas/ver/index.html)

## Ejemplo

```ts
const response = await factus.company.get();
const empresa = response.data;

console.log(empresa.graphic_representation_name);
console.log(empresa.nit);
console.log(empresa.email);
```
