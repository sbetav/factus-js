---
title: Ver empresa
description: Obtener los datos del perfil de la empresa autenticada en Factus.
---

## Firma del método

```ts
get(): Promise<ApiResponse<Company>>
```

## Endpoint Factus

`GET /v1/company`

Fuente oficial: [Ver empresa](https://developers.factus.com.co/empresas/ver/index.html)

## Parámetros

Este método no recibe parámetros.

## Ejemplo

```ts
const response = await factus.company.get();
const company = response.data;

console.log(company.graphic_representation_name);
console.log(company.nit);
console.log(company.email);
```
