## Métodos disponibles

| Método                        | Método HTTP + Endpoint    | Descripción                     |
| ----------------------------- | ------------------------- | ------------------------------- |
| `get(options?)`               | `GET /v2/companies`       | Obtener el perfil de la empresa |
| `update(input, options?)`     | `PUT /v2/companies`       | Actualizar el perfil            |
| `uploadLogo(image, options?)` | `POST /v2/companies/logo` | Subir o reemplazar el logotipo  |
