import { Callout } from "fumadocs-ui/components/callout";

## Firma del método

```ts
uploadLogo(image: File | Blob): Promise<ApiResponse<UploadCompanyLogoResponse>>
```

## Endpoint Factus

`POST /v1/company/logo`

Fuente oficial: [Actualizar logo de la empresa](https://developers.factus.com.co/empresas/actualizar-imagen/index.html)

## Parámetros

<TypeTable
  type={{
    image: {
      type: "File | Blob",
      required: true,
      description: "Archivo de imagen del logo en formato PNG, JPG o JPEG.",
    },
  }}
/>
## Ejemplo

```ts
import { readFileSync } from "fs";

const imageBuffer = readFileSync("./logo.png");
const imageBlob = new Blob([imageBuffer], { type: "image/png" });

const response = await factus.company.uploadLogo(imageBlob);
console.log(response.data.url_log);
```

<Callout type="info">
  La API retorna `url_log` (no `url_logo`) en este endpoint.
</Callout>

<Callout type="info">
  Este metodo no se puedde usar en entorno sandbox. Úsalo en producción o con
  credenciales autorizadas.
</Callout>
