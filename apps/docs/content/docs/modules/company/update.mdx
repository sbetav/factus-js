## Firma del método

```ts
update(
  input: UpdateCompanyInput,
  options?: RequestOptions,

```

## Endpoint Factus

`PUT /v2/companies`

Fuente oficial: [Actualizar empresa](https://developers.factus.com.co/empresas/actualizar)

## Ejemplo

```ts
const response = await factus.company.update({
  legal_organization_code: "2",
  company: null,
  names: "Alan",
  surnames: "Turing",
  responsibilities: "R-99-PN",
  economic_activity: "6920",
  email: "alanturing@enigmasas.com",
  address: "calle 100 #50-80",
  trade_name: null,
  registration_code: null,
  phone: "0987654321",
  municipality_code: "11001",
  tribute_code: "ZZ",
});

console.log(response.data.updated_at);
```
