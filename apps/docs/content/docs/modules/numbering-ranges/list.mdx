## Firma del método

```ts
list(
  params?: ListParams<NumberingRangeFilters>,
  options?: RequestOptions,
): Promise<ApiResponse<PaginatedData<NumberingRange>>>
```

## Endpoint Factus

`GET /v2/numbering-ranges`

Fuente oficial: [Listar rangos](https://developers.factus.com.co/rangos-de-numeracion/obtener-rangos)

## Ejemplo

```ts
const response = await factus.numberingRanges.list({
  filter: { is_active: "1" },
  page: 1,
  per_page: 10,
});

const numberingRanges = response.data.data;
const pagination = response.data.pagination;
```
