import { Callout } from "fumadocs-ui/components/callout";

## Firma del método

```ts
delete(id: number): Promise<DeleteNumberingRangeResponse>
```

## Endpoint Factus

`DELETE /v1/numbering-ranges/{numbering_range_id}`

Fuente oficial: [Eliminar rango](https://developers.factus.com.co/v1/rangos-de-numeracion/eliminar-rango/index.html)

<Callout type="warn">
  Si el rango no se ha usado, se elimina permanentemente. Si ya se usó, queda
  marcado como eliminado.
</Callout>

## Parámetros

<TypeTable
  type={{
    id: {
      type: "number",
      required: true,
      description: "ID numérico del rango de numeración a eliminar.",
    },
  }}
/>
## Ejemplo

```ts
await factus.numberingRanges.delete(8);
```

<Callout type="warn">
  Es probable que este método no funcione en entorno sandbox. Úsalo en
  producción o con credenciales autorizadas.
</Callout>
