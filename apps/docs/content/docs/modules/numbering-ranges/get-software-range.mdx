---
title: Rangos asociados al software
description: Obtener los rangos de numeración asociados al software.
---

## Firma del método

```ts
getSoftwareRange(): Promise<ApiResponse<SoftwareNumberingRange[]>>
```

## Endpoint Factus

`GET /v1/numbering-ranges/dian`

Fuente oficial: [Rangos asociados al software](https://developers.factus.com.co/v1/rangos-de-numeracion/rangos-asociados-al-software/index.html)

## Parámetros

Este método no recibe parámetros.

## Ejemplo

```ts
const response = await factus.numberingRanges.getSoftwareRange();
const numberingRanges = response.data;

for (const numberingRange of numberingRanges) {
  console.log(numberingRange.prefix, numberingRange.from, numberingRange.to);
}
```
