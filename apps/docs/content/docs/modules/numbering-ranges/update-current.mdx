import { Callout } from "fumadocs-ui/components/callout";

## Firma del método

```ts
updateCurrent(
  id: number,
  input: UpdateNumberingRangeCurrentInput,
  options?: RequestOptions,
): Promise<ApiResponse<NumberingRange>>
```

## Endpoint Factus

`PATCH /v2/numbering-ranges/{id}/current`

Fuente oficial: [Actualizar consecutivo](https://developers.factus.com.co/rangos-de-numeracion/actualizar-consecutivo)

## Ejemplo

```ts
const response = await factus.numberingRanges.updateCurrent(10, {
  current: 985000001,
});

console.log(response.data.current);
```
