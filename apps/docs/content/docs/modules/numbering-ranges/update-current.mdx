## Firma del método

```ts
updateCurrent(id: number, input: UpdateNumberingRangeCurrentInput): Promise<ApiResponse<NumberingRange>>
```

## Endpoint Factus

`PATCH /v1/numbering-ranges/{numbering_range_id}/current`

Fuente oficial: [Actualizar consecutivo](https://developers.factus.com.co/rangos-de-numeracion/actualizar-consecutivo/index.html)

## Ejemplo

```ts
const response = await factus.numberingRanges.updateCurrent(10, {
  current: 985000001,
});

console.log(response.data.current);
```
