---
title: Cambiar estado
description: Activar o desactivar un rango de numeración existente.
---

import { Callout } from "fumadocs-ui/components/callout";

## Firma del método

```ts
toggleStatus(
  id: number,
  options?: RequestOptions,
): Promise<ApiResponse<NumberingRange>>
```

## Endpoint Factus

`PATCH /v2/numbering-ranges/{id}/toggle-status`

Fuente oficial: [Cambiar estado](https://developers.factus.com.co/rangos-de-numeracion/cambiar-estado)

## Ejemplo

```ts
const response = await factus.numberingRanges.toggleStatus(8);

console.log(response.data.id);
console.log(response.data.is_active);
```

<Callout type="warn">
  Es probable que este método no funcione en entorno sandbox. Úsalo en
  producción o con credenciales autorizadas.
</Callout>
