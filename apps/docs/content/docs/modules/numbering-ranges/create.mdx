## Firma del método

```ts
create(
  input: CreateNumberingRangeInput,
  options?: RequestOptions,
): Promise<ApiResponse<NumberingRange>>
```

## Endpoint Factus

`POST /v2/numbering-ranges`

Fuente oficial: [Crear rango](https://developers.factus.com.co/rangos-de-numeracion/crear-rango)

## Ejemplo

```ts
import { NumberingRangeDocumentTypeCode } from "factus-js";

const response = await factus.numberingRanges.create({
  document: NumberingRangeDocumentTypeCode.SalesInvoice,
  prefix: "SETP",
  current: 984000000,
  resolution_number: "18760000009",
});

console.log(response.data.id);
```

<Callout type="warn">
  Es probable que este método no funcione en entorno sandbox. Úsalo en
  producción o con credenciales autorizadas.
</Callout>
