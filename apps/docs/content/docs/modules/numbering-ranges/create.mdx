## Firma del método

```ts
create(input: CreateNumberingRangeInput): Promise<ApiResponse<NumberingRange>>
```

## Endpoint Factus

`POST /v1/numbering-ranges`

Fuente oficial: [Crear rango](https://developers.factus.com.co/rangos-de-numeracion/crear-rango/index.html)

## Ejemplo

```ts
import { NumberingRangeDocumentTypeCode } from "factus-js";

const response = await factus.numberingRanges.create({
  document: NumberingRangeDocumentTypeCode.SalesInvoice,
  prefix: "SETP",
  current: 984000000,
  resolution_number: "18760000009",
});

console.log(response.data.id);
```
