## Firma del método

```ts
get(id: number): Promise<ApiResponse<NumberingRange>>
```

## Endpoint Factus

`GET /v1/numbering-ranges/{numbering_range_id}`

Fuente oficial: [Ver rango](https://developers.factus.com.co/rangos-de-numeracion/ver-rango/index.html)

## Parámetros

<TypeTable
  type={{
    id: {
      type: "number",
      required: true,
      description: "ID numérico del rango de numeración a consultar.",
    },
  }}
/>
## Ejemplo

```ts
const response = await factus.numberingRanges.get(4);
const numberingRange = response.data;

console.log(numberingRange.prefix);
console.log(numberingRange.current);
console.log(numberingRange.document);
```

<Callout type="warn">
  Es probable que este método no funcione en entorno sandbox. Úsalo en
  producción o con credenciales autorizadas.
</Callout>
