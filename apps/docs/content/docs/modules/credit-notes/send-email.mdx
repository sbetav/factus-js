---
title: Enviar correo
description: Enviar una nota crédito por correo usando el numero del documento.
---

import { Callout } from "fumadocs-ui/components/callout";

## Firma del método

```ts
sendEmail(number: string, input: SendCreditNoteEmailInput): Promise<SendCreditNoteEmailResponse>
```

## Endpoint Factus

`POST /v1/credit-notes/send-email/{number}`

Fuente oficial: [Enviar correo](https://developers.factus.com.co/notas-credito/enviar-correo/index.html)

## Parámetros

<TypeTable
  type={{
    number: {
      type: "string",
      required: true,

    },

    input: {
      type: "SendCreditNoteEmailInput (object)",

          Datos para enviar el correo electrónico.
          <TypeTable

                type: "string",

              },
            pdf_base_64_encoded: {
              type: "string",

      ),

/>

## Ejemplo

```ts
await factus.creditNotes.sendEmail("NC856", {
  email: "alanturing@enigmasas.com",
});
```

<Callout type="warn">
  Es probable que este método no funcione en entorno sandbox. Úsalo en
  producción o con credenciales autorizadas.
</Callout>
