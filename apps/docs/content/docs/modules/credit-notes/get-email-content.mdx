title: Obtener contenido de correo
description: Obtener asunto y adjunto base64 para envío manual de correo.

## Firma del método

```ts
getEmailContent(number: string): Promise<GetCreditNoteEmailContentResponse>
```

## Endpoint Factus

`GET /v1/credit-notes/{number}/email-content`

Fuente oficial: [Obtener contenido de correo](https://developers.factus.com.co/notas-credito/obtener-contenido-de-correo/index.html)

## Parámetros

<TypeTable
  type={{
    number: {
      type: "string",
      required: true,
      description:
        "Número del documento de la nota crédito para construir el contenido de correo.",
    },
  }}
/>
## Ejemplo

```ts
const response = await factus.creditNotes.getEmailContent("NC856");
console.log(response.data.subject);
```
