---
title: Ver nota crédito
description: Obtener el detalle completo de una nota crédito por numero.
---

## Firma del método

```ts
get(
  number: string,
  options?: RequestOptions,
): Promise<ViewCreditNoteResponse>
```

## Endpoint Factus

`GET /v2/credit-notes/{number}`

Fuente oficial: [Ver nota crédito](https://developers.factus.com.co/notas-credito/ver)

## Parámetros

<TypeTable
  type={{
    number: {
      type: "string",
      required: true,
      description: "Número del documento de la nota crédito.",
    },
  }}
/>
## Ejemplo

```ts
const response = await factus.creditNotes.get("NC856");
console.log(response.data.number);
console.log(response.data.totals?.total);
```
