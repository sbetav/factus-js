---
title: Ver nota crédito
description: Obtener el detalle completo de una nota crédito por numero.
---

## Firma

```ts
get(number: string): Promise<ApiResponse<ViewCreditNoteData>>
```

## Endpoint Factus

`GET /v1/credit-notes/{number}`

Fuente oficial: [Ver nota crédito](https://developers.factus.com.co/notas-credito/ver/index.html)

## Ejemplo

```ts
const response = await factus.creditNotes.get("NC72");
console.log(response.data.credit_note.total);
```
