## Métodos disponibles

| Método                                          | Método HTTP + Endpoint                                         | Descripción                                   |
| ----------------------------------------------- | -------------------------------------------------------------- | --------------------------------------------- |
| `create(input, options?)`                       | `POST /v2/credit-notes/validate`                               | Crear y validar una nota crédito              |
| `list(params?, options?)`                       | `GET /v2/credit-notes`                                         | Listar notas crédito con filtros y paginación |
| `listAll(filter?, options?)`                    | `GET /v2/credit-notes`                                         | Recorrer todas las páginas automáticamente    |
| `get(number, options?)`                         | `GET /v2/credit-notes/{number}`                                | Obtener el detalle de una nota crédito        |
| `downloadXml(number, options?)`                 | `GET /v2/credit-notes/{number}/download-xml`                   | Descargar el XML firmado                      |
| `downloadAttachedDocumentXml(number, options?)` | `GET /v2/credit-notes/{number}/download-attached-document-xml` | Descargar el AttachedDocument XML             |
| `downloadPdf(number, options?)`                 | `GET /v2/credit-notes/{number}/download-pdf`                   | Descargar la representación PDF               |
| `getEmailContent(number, options?)`             | `GET /v2/credit-notes/{number}/email-content`                  | Obtener asunto y adjunto para envío manual    |
| `sendEmail(number, input, options?)`            | `POST /v2/credit-notes/{number}/send-email`                    | Enviar la nota crédito por correo             |
| `delete(referenceCode, options?)`               | `DELETE /v2/credit-notes/reference/{reference_code}`           | Eliminar una nota crédito no validada         |

Consulta [Paginación automática](/docs/pagination) para el uso de `listAll()`.
