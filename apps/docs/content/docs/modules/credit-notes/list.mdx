title: Ver y filtrar notas crédito
description: Listar notas crédito con filtros y paginación.
---

## Firma

```ts
list(filters?: CreditNoteFilters & { page?: number; per_page?: number }): Promise<GetCreditNotesResponse>
```

## Endpoint Factus

`GET /v1/credit-notes`

Fuente oficial: [Ver y filtrar notas crédito](https://developers.factus.com.co/notas-credito/ver-filtrar/index.html)

## Filtros

- `filter[identification]`
- `filter[names]`
- `filter[number]`
- `filter[prefix]`
- `filter[reference_code]`
- `filter[status]` (1 validada, 0 pendiente)

## Ejemplo

```ts
const response = await factus.creditNotes.list({
  "filter[status]": "1",
  page: 1,
  per_page: 10,
});

const notas = response.data.data;
const paginacion = response.data.pagination;
```
