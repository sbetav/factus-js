## Firma del método

```ts
list(params?: ListParams<CreditNoteFilters>): Promise<GetCreditNotesResponse>
```

## Endpoint Factus

`GET /v1/credit-notes`

Fuente oficial: [Ver y filtrar notas crédito](https://developers.factus.com.co/v1/notas-credito/ver-filtrar/index.html)

## Ejemplo

```ts
const response = await factus.creditNotes.list({
  filter: { status: "1" },
  page: 1,
  per_page: 10,
});

const creditNotes = response.data.data;
const pagination = response.data.pagination;
```
