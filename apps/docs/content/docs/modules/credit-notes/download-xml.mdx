---
title: Descargar XML
description: Descargar XML de una nota crédito en base64 por numero.
---

## Firma del método

```ts
downloadXml(
  number: string,
  options?: RequestOptions,
): Promise<DownloadCreditNoteXmlResponse>
```

## Endpoint Factus

`GET /v2/credit-notes/{number}/download-xml`

Fuente oficial: [Descargar XML](https://developers.factus.com.co/notas-credito/descargar-xml)

## Parámetros

<TypeTable
  type={{
    number: {
      type: "string",
      required: true,
      description:
        "Número del documento de la nota crédito cuyo XML se descargará.",
    },
  }}
/>
## Ejemplo

```ts
const response = await factus.creditNotes.downloadXml("NC856");
const xml = Buffer.from(response.data.xml_base_64_encoded, "base64").toString(
  "utf-8",
);
```
