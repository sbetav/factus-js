## Firma del método

```ts
delete(
  referenceCode: string,
  options?: RequestOptions,
): Promise<DeleteCreditNoteResponse>
```

## Endpoint Factus

`DELETE /v2/credit-notes/reference/{reference_code}`

Fuente oficial: [Eliminar nota crédito](https://developers.factus.com.co/notas-credito/eliminar)

<Callout type="warn">
  Solo se pueden eliminar notas crédito que no hayan sido validadas por la DIAN.
</Callout>

```ts
await factus.creditNotes.delete("NC856");
```
