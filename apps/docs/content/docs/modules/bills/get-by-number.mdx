---
title: Ver factura
description: Obtener el detalle completo de una factura por su número de documento.
---

## Firma del método

```ts
getByNumber(number: string): Promise<ViewInvoiceResponse>
```

## Endpoint Factus

`GET /v1/bills/show/{number}`

Fuente oficial: [Ver factura](https://developers.factus.com.co/facturas/ver/index.html)

## Parámetros

<TypeTable
  type={{
    number: {
      type: "string",
      required: true,
      description: (
        <>
          {"Número del documento de la factura (por ejemplo "}
          <code>{"SETP990000203"}</code>
          {")."}
        </>
      ),
    },
  }}
/>
## Ejemplo

```ts
const response = await factus.bills.getByNumber("SETP990000203");
console.log(response.data);
```
