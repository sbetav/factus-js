## Firma del método

```ts
create(
  input: CreateBillInput,
  options?: RequestOptions,
): Promise<CreateBillResponse>
```

## Endpoint Factus

`POST /v2/bills/validate`

Fuente oficial: [Crear y validar](https://developers.factus.com.co/facturas/crear-y-validar)
