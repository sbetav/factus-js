## Firma del método

```ts
delete(
  referenceCode: string,
  options?: RequestOptions,
): Promise<DeleteBillResponse>
```

## Endpoint Factus

`DELETE /v2/bills/destroy/reference/{reference_code}`

Fuente oficial: [Eliminar factura](https://developers.factus.com.co/facturas/eliminar)

<Callout type="warn">
  Solo se pueden eliminar facturas que no hayan sido validadas por la DIAN.
</Callout>

## Parámetros

<TypeTable
  type={{

## Ejemplo

```ts
await factus.bills.delete("SETP990000049");
```
