title: Eliminar factura
description: Eliminar una factura no validada por la DIAN usando reference_code.

import { Callout } from "fumadocs-ui/components/callout";

## Firma del método

```ts
delete(referenceCode: string): Promise<DeleteInvoiceResponse>
```

## Endpoint Factus

`DELETE /v1/bills/destroy/reference/{reference_code}`

Fuente oficial: [Eliminar factura no validada](https://developers.factus.com.co/facturas/eliminar/index.html)

<Callout type="warn">
  Solo se pueden eliminar facturas que no hayan sido validadas por la DIAN.
</Callout>

## Parámetros

<TypeTable
  type={{
    referenceCode: {
      type: "string",
      required: true,
      description: (
        <>
          {"Código de referencia ("}

/>
## Ejemplo

```ts
await factus.bills.delete("SETP990000049");
```

<Callout type="warn">
  Es probable que este método no funcione en entorno sandbox.
