---
title: Obtener eventos RADIAN
description: Obtener la lista de eventos RADIAN registrados en una factura.
---

## Firma del método

```ts
getEvents(
  number: string,
  options?: RequestOptions,
): Promise<GetBillEventsResponse>
```

## Endpoint Factus

`GET /v2/bills/{number}/radian/events`

Fuente oficial: [Eventos de facturas](https://developers.factus.com.co/facturas/eventos-de-facturas)

## Parámetros

<TypeTable
  type={{

  }}
/>
## Ejemplo

```ts
const response = await factus.bills.getEvents("SETP990000203");
console.log(response.data);
```
