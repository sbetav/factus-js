---
title: Obtener eventos RADIAN
description: Obtener la lista de eventos RADIAN registrados en una factura.
---

## Firma del método

```ts
getEvents(number: string): Promise<GetBillEventsResponse>
```

## Endpoint Factus

`GET /v1/bills/{number}/radian/events`

Fuente oficial: [Eventos de facturas](https://developers.factus.com.co/v1/facturas/eventos-de-facturas/index.html)

## Parámetros

<TypeTable
  type={{
    number: {
      type: "string",
      required: true,
      description:
        "Número del documento de la factura para consultar sus eventos RADIAN.",
    },
  }}
/>
## Ejemplo

```ts
const response = await factus.bills.getEvents("SETP990000203");
console.log(response.data);
```
