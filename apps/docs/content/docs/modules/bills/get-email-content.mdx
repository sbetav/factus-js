---
title: Obtener contenido de correo
description: Obtener asunto y adjunto (base64) para envío manual de correo.
---

## Firma del método

```ts
getEmailContent(number: string): Promise<GetBillEmailContentResponse>
```

## Endpoint Factus

`GET /v1/bills/{number}/email-content`

Fuente oficial: [Obtener contenido de correo](https://developers.factus.com.co/v1/facturas/obtener-contenido-de-correo/index.html)

## Parámetros

<TypeTable
  type={{
    number: {
      type: "string",
      required: true,
      description:
        "Número del documento de la factura para construir el contenido de correo.",
    },
  }}
/>
## Ejemplo

```ts
const response = await factus.bills.getEmailContent("SETP990000203");
console.log(response.data.subject);
```
