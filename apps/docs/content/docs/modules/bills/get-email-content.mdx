---
title: Obtener contenido de correo
description: Obtener asunto y adjunto (base64) para envío manual de correo.
---

## Firma del método

```ts
getEmailContent(number: string): Promise<GetInvoiceEmailContentResponse>
```

## Endpoint Factus

`GET /v1/bills/{number}/email-content`

Fuente oficial: [Obtener contenido de correo](https://developers.factus.com.co/facturas/obtener-contenido-de-correo/index.html)

## Ejemplo

```ts
const response = await factus.bills.getEmailContent("SETP990000203");
console.log(response.data.subject);
```
