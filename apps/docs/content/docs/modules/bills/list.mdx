## Firma del método

```ts
list(
  params?: ListParams<BillFilters>,
  options?: RequestOptions,
): Promise<GetBillsResponse>
```

## Endpoint Factus

`GET /v2/bills`

Fuente oficial: [Listar facturas](https://developers.factus.com.co/facturas/listar-y-filtrar)

## Ejemplo

```ts
const response = await factus.bills.list({
  filter: {
    status: "1",
    number: "SETP99",
  },
  page: 1,
  per_page: 15,
});

const facturas = response.data.data;
const paginacion = response.data.pagination;
```
