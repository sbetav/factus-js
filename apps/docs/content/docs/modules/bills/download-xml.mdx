---
title: Descargar XML
description: Descargar XML de una factura en base64 usando el numero del documento.
---

## Firma del método

```ts
downloadXml(number: string): Promise<DownloadInvoiceXmlResponse>
```

## Endpoint Factus

`GET /v1/bills/download-xml/{number}`

Fuente oficial: [Descargar XML](https://developers.factus.com.co/facturas/descargar-xml/index.html)

## Parámetros

<TypeTable
  type={{
    "number": {
      type: "string",
      required: true,
      description: "Número del documento de la factura cuyo XML se descargará.",
    },
  }}
/>
## Ejemplo

```ts
const response = await factus.bills.downloadXml("SETP990000203");
const xml = Buffer.from(response.data.xml_base_64_encoded, "base64").toString(
  "utf-8",
);
```
