---
title: Ver factura
description: Obtener el detalle completo de una factura por su número de documento.
---

## Firma del método

```ts
get(number: string, options?: RequestOptions): Promise<ViewInvoiceResponse>
```

## Endpoint Factus

`GET /v1/bills/show/{number}`

Fuente oficial: [Ver factura](https://developers.factus.com.co/facturas/ver/index.html)

## Parámetros

<TypeTable
  type={{
    number: {
      type: "string",
      required: true,
      description: (
        <>
          {"Número del documento de la factura (por ejemplo "}
          <code>{"SETP990000203"}</code>
          {")."}
        </>
      ),
    },
    options: {
      type: "RequestOptions",
      required: false,
      description: "Opciones opcionales de la solicitud (p. ej. signal para AbortSignal).",
    },
  }}
/>

## Ejemplo

```ts
const response = await factus.bills.get("SETP990000203");
console.log(response.data);
```
