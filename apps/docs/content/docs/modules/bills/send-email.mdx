title: Enviar correo
description: Enviar por correo electrónico una factura usando su numero.

import { Callout } from "fumadocs-ui/components/callout";

## Firma del método

```ts
sendEmail(
  number: string,
  input: SendBillEmailInput,
  options?: RequestOptions,
): Promise<SendBillEmailResponse>
```

## Endpoint Factus

`POST /v2/bills/{number}/send-email`

Fuente oficial: [Enviar correo](https://developers.factus.com.co/facturas/enviar-correo)

## Parámetros

## Ejemplo

```ts
await factus.bills.sendEmail("SETP990000203", {
  email: "alanturing@enigmasas.com",
});
```
