---
title: Enviar correo
description: Enviar por correo electrónico una factura usando su numero.
---

import { Callout } from "fumadocs-ui/components/callout";

## Firma del método

```ts
sendEmail(number: string, input: SendBillEmailInput): Promise<SendBillEmailResponse>
```

## Endpoint Factus

`POST /v1/bills/send-email/{number}`

Fuente oficial: [Enviar correo](https://developers.factus.com.co/facturas/enviar-correo/index.html)

## Parámetros

<TypeTable
  type={{
    number: {
      type: "string",
      required: true,
      description: "Número del documento de la factura a enviar.",
    },
    input: {
      type: "SendBillEmailInput (object)",
      required: true,
      description: (
        <div>
          Datos para enviar el correo electrónico.
          <TypeTable
            type={{
              email: {
                type: "string",
                required: true,
                description: "Correo destinatario.",
              },
            }}
          />
        </div>
      ),
    },
  }}
/>

## Ejemplo

```ts
await factus.bills.sendEmail("SETP990000203", {
  email: "alanturing@enigmasas.com",
});
```

<Callout type="warn">
  Es probable que este método no funcione en entorno sandbox. Úsalo en
  producción o con credenciales autorizadas.
</Callout>
