---
title: Obtener adquiriente
description: Consultar nombre y correo del adquiriente por tipo y número de documento.
---

## Firma del método

```ts
get(
  filters: AcquirerFilters,
  options?: RequestOptions,
): Promise<ApiResponse<Acquirer>>
```

## Endpoint Factus

`GET /v2/dian/acquirer`

Fuente oficial: [Obtener datos de adquiriente](https://developers.factus.com.co/informacion-adquirientes/obtener-datos-adquiriente)

<TypeTable
  type={{

      type: "AcquirerFilters (object)",

        <div>

          <TypeTable
            type={{
              identification_document_code: {

              identification_number: {
                type: "string",

## Ejemplo

```ts
import { IdentityDocumentCode } from "factus-js";

const response = await factus.acquirer.get({
  identification_document_code: IdentityDocumentCode.NIT,
  identification_number: "1399991",
});

console.log(response.data.name);
console.log(response.data.email);
```
