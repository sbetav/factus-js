## Requisitos

- **Node.js** 18 o superior (se requiere la API `fetch` nativa).
- Una cuenta activa en [Factus](https://factus.com.co) con credenciales de API.

## Instalar el paquete principal

La forma recomendada es instalar `factus-js`, el paquete de entrada que reexporta todo — el cliente, los tipos y las constantes.

<Tabs items={['npm', 'yarn', 'pnpm', 'bun']}>

<Tab value="npm">

```bash
npm install factus-js
```

</Tab>

<Tab value="yarn">

```bash
yarn add factus-js
```

</Tab>

<Tab value="pnpm">

```bash
pnpm add factus-js
```

</Tab>

<Tab value="bun">

```bash
bun add factus-js
```

</Tab>

</Tabs>

## Paquetes del monorepo

| Paquete                | Descripción                                                                 |
| ---------------------- | --------------------------------------------------------------------------- |
| `factus-js`            | Punto de entrada principal — reexporta todo desde los paquetes internos     |
| `@factus-js/client`    | Clase `FactusClient` con todos los recursos de la API                       |
| `@factus-js/types`     | Interfaces TypeScript para entradas, salidas y respuestas                   |
| `@factus-js/constants` | Objetos de constantes tipadas (códigos de documento, métodos de pago, etc.) |

<Callout type="info">
  `factus-js` reexporta todo desde `@factus-js/client`, `@factus-js/types` y
  `@factus-js/constants`. No necesitas instalar los paquetes internos por
  separado a menos que quieras dependencias granulares.
</Callout>

## Instalación avanzada (paquetes individuales)

Si prefieres dependencias más granulares, puedes instalar los paquetes internos directamente:

<Tabs items={['npm', 'yarn', 'pnpm', 'bun']}>

<Tab value="npm">

```bash
npm install @factus-js/client @factus-js/types
# opcional — constantes tipadas
npm install @factus-js/constants
```

</Tab>

<Tab value="yarn">

```bash
yarn add @factus-js/client @factus-js/types
# opcional — constantes tipadas
yarn add @factus-js/constants
```

</Tab>

<Tab value="pnpm">

```bash
pnpm add @factus-js/client @factus-js/types
# opcional — constantes tipadas
pnpm add @factus-js/constants
```

</Tab>

<Tab value="bun">

```bash
bun add @factus-js/client @factus-js/types
# opcional — constantes tipadas
bun add @factus-js/constants
```

</Tab>

</Tabs>

## Variables de entorno

Guarda tus credenciales en variables de entorno. Nunca las escribas directamente en el código fuente.

```bash
# .env
FACTUS_CLIENT_ID=tu-client-id
FACTUS_CLIENT_SECRET=tu-client-secret
FACTUS_USERNAME=tu@email.com
FACTUS_PASSWORD=tu-contraseña
```

A continuación, consulta el [inicio rápido](/docs/quick-start) para ver cómo inicializar el cliente y hacer tu primera llamada.
