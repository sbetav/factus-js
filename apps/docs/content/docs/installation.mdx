## Requisitos

- **Node.js** 18 o superior (se requiere la API `fetch` nativa).
- Una cuenta activa en [Factus](https://factus.com.co) con credenciales de API.

## Instalar el paquete

Instala `factus-js`, que incluye cliente, tipos y constantes tipadas.

<Tabs items={['npm', 'yarn', 'pnpm', 'bun']}>

<Tab value="npm">

```bash
npm install factus-js
```

</Tab>

<Tab value="yarn">

```bash
yarn add factus-js
```

</Tab>

<Tab value="pnpm">

```bash
pnpm add factus-js
```

</Tab>

<Tab value="bun">

```bash
bun add factus-js
```

</Tab>

</Tabs>

## Paquete del SDK

| Paquete     | Descripción                                                      |
| ----------- | ---------------------------------------------------------------- |
| `factus-js` | SDK principal con `FactusClient`, tipos TypeScript y constantes. |

## Variables de entorno

Guarda tus credenciales en variables de entorno. Nunca las escribas directamente en el código fuente.

```bash
# .env
FACTUS_CLIENT_ID=tu-client-id
FACTUS_CLIENT_SECRET=tu-client-secret
FACTUS_USERNAME=tu@email.com
FACTUS_PASSWORD=tu-contraseña
```

A continuación, consulta el [inicio rápido](/docs/quick-start) para ver cómo inicializar el cliente y hacer tu primera llamada.
