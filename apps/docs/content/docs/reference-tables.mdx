Esta guía consolida las tablas oficiales de Factus y su equivalente en
constantes de `factus-js`.

Fuente oficial:
[Tablas de referencia](https://developers.factus.com.co/tablas-de-referencia/tablas)

## Modo de uso

Si instalaste `factus-js`, las constantes ya están disponibles. Cada constante es
un mapa de claves a valores string, por lo que puedes usarlas directamente en
los campos de la API:

```ts
import { PaymentFormCode, EventCode } from "factus-js";

const paymentForm = PaymentFormCode.CashPayment; // "1"
const event = EventCode.ExpressAcceptance; // "033"
```

<Callout type="info">
  Las constantes se exportan desde la raíz del paquete y no requieren llamadas
  HTTP.
</Callout>

## Datos de referencia externos

Factus documenta países, municipios y unidades de medida como tablas de
referencia externas. El SDK no expone endpoints runtime para esos datasets.

- [Países](https://developers.factus.com.co/tablas-de-referencia/countries/)
- [Municipios](https://developers.factus.com.co/tablas-de-referencia/municipios/)
- [Unidades de medida](https://developers.factus.com.co/tablas-de-referencia/unit-measures/)

## Descripciones y abreviaturas (`*Info`)

Para mostrar etiquetas legibles en UI, importa los mapas `*Info`
complementarios desde `factus-js`:

```ts
import {
  PaymentFormCode,
  PaymentFormCodeInfo,
  IdentityDocumentCode,
  IdentityDocumentCodeInfo,
} from "factus-js";

const desc = PaymentFormCodeInfo[PaymentFormCode.CashPayment].description;
const abbr =
  IdentityDocumentCodeInfo[IdentityDocumentCode.CitizenshipCard].abbreviation;
```
