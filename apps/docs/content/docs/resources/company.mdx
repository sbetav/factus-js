# Empresa (`company`)

El recurso `company` gestiona el **perfil de la empresa** registrada en Factus. Se accede a través de `factus.company`.

## Métodos disponibles

| Método              | Verbo + Ruta            | Descripción                                |
| ------------------- | ----------------------- | ------------------------------------------ |
| `get()`             | `GET /v1/company`       | Obtener los datos del perfil de la empresa |
| `update(input)`     | `PUT /v1/company`       | Actualizar los datos del perfil            |
| `uploadLogo(image)` | `POST /v1/company/logo` | Subir el logotipo de la empresa            |

---

## `get()`

Obtiene los datos del perfil de la empresa actualmente autenticada.

```ts
get(): Promise<ApiResponse<Company>>
```

```ts
const response = await factus.company.get();
const empresa = response.data;

console.log(empresa.graphic_representation_name); // "Mi Empresa S.A.S."
console.log(empresa.nit);
console.log(empresa.email);
```

---

## `update(input)`

Actualiza los datos del perfil de la empresa.

```ts
update(input: UpdateCompanyInput): Promise<ApiResponse<Company>>
```

```ts
const response = await factus.company.update({
  legal_organization_code: "1",
  company: "Mi Empresa Actualizada S.A.S.",
  trade_name: "Mi Empresa",
  economic_activity: "6201",
  email: "nuevo@empresa.com",
  phone: "6011234567",
  address: "Av. El Dorado # 92-32",
  municipality_code: "11001",
  tribute_code: "ZZ",
  responsibilities: "5",
});

console.log(response.data.graphic_representation_name);
```

## `uploadLogo(image)`

Sube el logotipo de la empresa. Formatos aceptados: PNG, JPG, JPEG.

```ts
uploadLogo(image: File | Blob): Promise<ApiResponse<UploadCompanyLogoResponse>>
```

```ts
// En Node.js (leer desde archivo)
import { readFileSync } from "fs";

const imageBuffer = readFileSync("./logo.png");
const imageBlob = new Blob([imageBuffer], { type: "image/png" });

const response = await factus.company.uploadLogo(imageBlob);
console.log(response.data.url_log); // URL del logotipo subido
```

```ts
// En el navegador (desde un <input type="file">)
const input = document.querySelector<HTMLInputElement>('input[type="file"]');
const file = input?.files?.[0];

if (file) {
  const response = await factus.company.uploadLogo(file);
  console.log(response.data.url_log);
}
```

> **Nota:** La API retorna la clave `url_log` (no `url_logo`) en la respuesta. El tipo `UploadCompanyLogoResponse` refleja esto exactamente.
