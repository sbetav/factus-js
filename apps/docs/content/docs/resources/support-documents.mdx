# Documentos soporte (`supportDocuments`)

El recurso `supportDocuments` gestiona los **documentos soporte** electrónicos, requeridos por la DIAN para respaldar compras a personas o entidades no obligadas a expedir factura. Se accede a través de `factus.supportDocuments`.

<Callout type="info">
  Para la especificación completa de campos y reglas de validación de la DIAN,
  consulta la [documentación oficial de
  Factus](https://developers.factus.com.co/).
</Callout>

## Métodos disponibles

| Método                | Verbo + Ruta                                  | Descripción                                      |
| --------------------- | --------------------------------------------- | ------------------------------------------------ |
| `create(input)`       | `POST /v1/support-documents/validate`         | Crear y validar un nuevo documento soporte       |
| `list(filters?)`      | `GET /v1/support-documents`                   | Listar documentos soporte con filtros opcionales |
| `get(id)`             | `GET /v1/support-documents/{id}`              | Obtener detalle por ID numérico                  |
| `getByNumber(number)` | `GET /v1/support-documents/show/{number}`     | Obtener detalle por número de documento          |
| `downloadXml(id)`     | `GET /v1/support-documents/download-xml/{id}` | Descargar XML en base64                          |
| `downloadPdf(id)`     | `GET /v1/support-documents/download-pdf/{id}` | Descargar PDF en base64                          |
| `delete(id)`          | `DELETE /v1/support-documents/{id}`           | Eliminar un documento soporte no validado        |

## `create(input)`

```ts
create(input: CreateSupportDocumentInput): Promise<ApiResponse<SupportDocument>>
```

```ts
import { SupportDocumentIdentityTypeId } from "@factus-js/constants";

const response = await factus.supportDocuments.create({
  numbering_range_id: 125,
  reference_code: "DS-001",
  observation: "Compra a proveedor no obligado a facturar",
  payment_method_code: "10",
  provider: {
    identification: "12345678",
    names: "Pedro Proveedor",
    address: "Carrera 10 # 20-30",
    email: "proveedor@correo.com",
    phone: "3109876543",
    country_code: "CO",
    identification_document_id: SupportDocumentIdentityTypeId.NIT,
    municipality_id: 149,
  },
  items: [
    {
      code_reference: "SRV-001",
      name: "Servicio prestado",
      quantity: 1,
      discount_rate: 0,
      price: 200000,
      unit_measure_id: 70,
      standard_code_id: 1,
    },
  ],
});

console.log(response.data.number);
```

<Callout type="info">
  Los documentos soporte tienen un subconjunto restringido de tipos de
  identificación de proveedor. Usa `SupportDocumentIdentityTypeId` del paquete
  `@factus-js/constants`.
</Callout>

### Tipos de identificación para documentos soporte (`SupportDocumentIdentityTypeId`)

| Constante                         | Valor | Descripción                            |
| --------------------------------- | ----- | -------------------------------------- |
| `FOREIGNER_CARD`                  | `4`   | Tarjeta de extranjería                 |
| `FOREIGNER_ID`                    | `5`   | Cédula de extranjería                  |
| `NIT`                             | `6`   | NIT                                    |
| `PASSPORT`                        | `7`   | Pasaporte                              |
| `FOREIGN_IDENTIFICATION_DOCUMENT` | `8`   | Documento de identificación extranjero |
| `PEP`                             | `9`   | Permiso especial de permanencia        |
| `FOREIGN_NIT`                     | `10`  | NIT de otro país                       |

## `list(filters?)`

```ts
list(filters?: SupportDocumentFilters & { page?: number; per_page?: number }): Promise<ApiResponse<PaginatedData<SupportDocument>>>
```

```ts
const response = await factus.supportDocuments.list({ page: 1, per_page: 10 });
const documentos = response.data.data;
const paginacion = response.data.pagination;
```

## `delete(id)`

<Callout type="warn">
  Solo se pueden eliminar documentos soporte que no hayan sido validados por la
  DIAN.
</Callout>

```ts
delete(id: number): Promise<DeleteSupportDocumentResponse>
```

```ts
await factus.supportDocuments.delete(321);
```
