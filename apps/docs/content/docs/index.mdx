# factus-js

**factus-js** es un SDK de Javascript para la [API de Factus](https://factus.com.co), una plataforma colombiana de facturación electrónica homologada por la DIAN. El SDK proporciona una capa de abstracción completa sobre la API REST, con autenticación OAuth2 automática y tipado estricto en todos los métodos.

<Callout type="info">
  Para más información sobre la API y sus endpoints visita la [documentación
  oficial de Factus](https://developers.factus.com.co/).
</Callout>

## Paquete

El SDK se distribuye como un único paquete en npm:

| Paquete     | Descripción                                                                     |
| ----------- | ------------------------------------------------------------------------------- |
| `factus-js` | Cliente principal (`FactusClient`) + tipos TypeScript + constantes tipadas DIAN |
