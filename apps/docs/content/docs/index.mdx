# factus-js

**factus-js** es un SDK de TypeScript para la [API de Factus](https://factus.com.co), la plataforma colombiana de facturación electrónica homologada por la DIAN. El SDK proporciona una capa de abstracción completa sobre la API REST, con autenticación OAuth2 automática y tipado estricto en todos los métodos.

<Callout type="info">
  Para la documentación oficial y completa de la API de Factus, visita la
  [documentación oficial de Factus](https://developers.factus.com.co/).
</Callout>

## Paquetes

El repositorio es un monorepo con tres paquetes publicados en npm:

| Paquete                | Descripción                                                                     |
| ---------------------- | ------------------------------------------------------------------------------- |
| `@factus-js/client`    | Cliente principal — `FactusClient` y todos los recursos                         |
| `@factus-js/types`     | Interfaces TypeScript para todas las entidades y respuestas                     |
| `@factus-js/constants` | Constantes con tipado — códigos DIAN, tipos de documento, métodos de pago, etc. |

## Inicio rápido

```ts
import { FactusClient } from "@factus-js/client";

const factus = new FactusClient({
  clientId: process.env.FACTUS_CLIENT_ID!,
  clientSecret: process.env.FACTUS_CLIENT_SECRET!,
  username: process.env.FACTUS_USERNAME!,
  password: process.env.FACTUS_PASSWORD!,
  environment: "sandbox",
});

const response = await factus.invoices.create({
  /* ... */
});
console.log(response.data.number); // "SETP990000001"
```

Consulta la [guía de instalación](/docs/getting-started/installation) y el [inicio rápido](/docs/getting-started/quick-start) para más detalles.
