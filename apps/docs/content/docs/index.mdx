## factus-js

**factus-js** es un SDK para JavaScript y TypeScript que simplifica la integración de la [API de Factus](https://factus.com.co) en aplicaciones Node.js. El SDK proporciona autenticación OAuth2 automática, tipado estricto y una capa consistente para trabajar con facturas electrónicas, notas crédito, documentos soporte y más.

<Callout type="info">
  Este no es un producto oficial de Factus. Para más información visita la
  [documentación oficial](https://developers.factus.com.co/).
</Callout>

<Callout type="warn">
  Esta versión del SDK utiliza la **API v2** de Factus, asegúrate de que tus
  credenciales estén habilitadas para usar este entorno — Si lo necesitas puedes
  encontrar la versión anterior del SDK [aquí](https://factusjs-v1.vercel.app).
</Callout>

## Paquete

El SDK se distribuye como un único paquete en npm:

| Paquete     | Descripción                                                                     |
| ----------- | ------------------------------------------------------------------------------- |
| `factus-js` | Cliente principal (`FactusClient`) + tipos TypeScript + constantes tipadas DIAN |

## Qué resuelve este SDK

- Integrar Factus API v2 utilizando JavaScript/TypeScript.
- Centralizar autenticación, refresh de tokens, timeouts, cancelación y manejo de errores.
- Proveer constantes tipadas para los códigos estables del API.
