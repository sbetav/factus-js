## factus-js

**factus-js** es un SDK para JavaScript y TypeScript que simplifica la integración de la [API de Factus](https://factus.com.co) en aplicaciones Node.js. El SDK proporciona autenticación OAuth2 automática, tipado estricto y una capa consistente para trabajar con facturas electrónicas, notas crédito, documentos soporte y catálogos DIAN.

<Callout type="info">
  Este no es un producto oficial de Factus. Para más información visita la
  [documentación oficial](https://developers.factus.com.co/v1/).
</Callout>

## Paquete

El SDK se distribuye como un único paquete en npm:

| Paquete     | Descripción                                                                     |
| ----------- | ------------------------------------------------------------------------------- |
| `factus-js` | Cliente principal (`FactusClient`) + tipos TypeScript + constantes tipadas DIAN |

## Qué resuelve este SDK

- Integrar Factus API desde Node.js con JavaScript o TypeScript.
- Emitir facturas electrónicas, notas crédito y documentos soporte.
- Usar constantes y tablas DIAN tipadas para reducir errores de validación.
- Centralizar autenticación, refresh de tokens y manejo de errores.
