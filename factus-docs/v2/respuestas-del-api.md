# Respuestas del API

#### **Códigos de respuesta HTTP**

[Sección titulada «Códigos de respuesta HTTP»](https://developers.factus.com.co/respuestas-del-api#c%C3%B3digos-de-respuesta-http)

El API puede retornar alguno de los siguientes códigos HTTP:

| Código | Descripción |
| --- | --- |
| `200` | Solicitud exitosa |
| `201` | Recurso creado exitosamente |
| `204` | Sin contenido |
| `400` | Solicitud mal formada |
| `401` | No autorizado / No autenticado |
| `402` | Pago requerido |
| `403` | Acceso denegado |
| `404` | Ruta o recurso no encontrado |
| `409` | Conflicto |
| `422` | Errores de validación |
| `429` | Demasiadas solicitudes |
| `500` | Error interno del servidor |
| `503` | Servicio no disponible / En mantenimiento |
