# Colección de Factus API v2

Esta colección está pensada para ayudarte a explorar y probar de forma rápida los endpoints de **Factus API v2** desde Bruno. Incluye solicitudes listas para ejecutar sobre los principales módulos de la API, como facturas, notas crédito, rangos de numeración y links a la documentación oficial.

Al importarla, tendrás una base práctica para validar tu integración, entender la estructura de las peticiones y revisar respuestas esperadas en distintos escenarios. Puedes adaptarla a tu flujo de trabajo y complementar cada solicitud con tus credenciales y datos de prueba.
