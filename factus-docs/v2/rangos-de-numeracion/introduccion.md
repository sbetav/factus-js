# Introducción

El endpoint permite obtener los rangos de numeración disponibles en la API de Factus. Este recurso es útil para obtener información precisa sobre los rangos de numeración de documentos específicos, incluyendo su prefijo, rango de numeración, resolución, fecha de inicio y fin, entre otros.

* * *

El rango de numeración es un parámetro que se debe usar al momento de crear un documento electrónico, este parámetro es requerido y se debe enviar el id del rango de numeración que se desea usar.
